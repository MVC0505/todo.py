This is a simple command-line To-Do List application written in Python. It allows users to add, remove, and view tasks. All tasks are stored in a text file, ensuring that the list persists between program runs.

Features

  Add new tasks

  Remove existing tasks

  View all tasks

  Persistent storage using a text file


How It Works

  Tasks are stored in tasks.txt

  The program loads tasks when it starts

  Any changes (add/remove) are saved automatically


How to Run

  Make sure Python is installed

  Save the script as todo.py

  Run the program using:

    python todo.py
